Instructions

The Facebook messages browser allows you to browse your archived Facebook threads, which you first need to download from your Facebook account (in JSON format)

To download your Facebook threads in JSON format, follow these instructions:

  • Go to your Facebook account settings page
  • Select the "Your personal information" tab from the left panel
  • Click on "Download your information"
  • Deselect all checkboxes except for "Messages". In the "Format" dropdown menu, select "JSON". Optionally, provide a date range.
  • Click on "Create a file"
  • Once it has been created, download the archive
  • Unpack the archive such that the messages/ folder is in the same directory as this file.
  • Locate the JSON of the threads you want to upload. Note: if there are multiple files for the same thread (e.g.: message_1.json, message_2.json, etc.), you can upload all of them at the same time to automatically merge them.
  • Upload them using the button in the left sidebar
