About the project

I initially started this side project because I wanted to browse my own Facebook threads, but found it hard to do so on Facebook, particularly since it lacks an option to go to a particular date. As the project grew, I thought it could interest other people as well, which is why I decided to put it online.

The browser makes it easier to view your threads, in particular by allowing you to go to any date using the date picker, and by saving your location in the thread when you leave the page, so that you can resume reading the thread from the same location the next time you open the page.

Right now, only message contents are shown - but I am planning on adding functionality in the near future, such as reactions, shared content, etc.

Finally, you will find some fun graphs about your threads in the analytics tab!

How it works

The Facebook messages browser is a pure JavaScript application: everything happens in the browser. There is no backend, and I want to emphasize that nothing gets uploaded. Basically, it is a "pretty" interface for the raw JSON thread, which I try to make visually close to the real Facebook messenger interface.

The threads get persisted using IndexedDB, so that you don't have to reupload them every time you use this website. The graphs in the analytics page are generated with Chart.js, and the date picker is created with Daterangepicker.
